## How can I edit this code?

There are several ways of editing your application.

**Use Lovable**

Simply visit the Lovable Project link shown in your project settings and start prompting.

Changes made via Lovable will be committed automatically to this repo.

**Use your preferred IDE**

If you want to work locally using your IDE, clone this repository and follow your normal development workflow.

The only requirement is having Node.js & npm installed. Use your preferred Node version manager (nvm, volta, asdf, etc.) to install Node.js.

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

You can edit files locally or use any cloud development environment. Follow the editor or service documentation to create, commit, and push changes.
